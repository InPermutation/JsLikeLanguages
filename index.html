Dart

  • Open-source language from Google
  • Intended to replace JS
  • dart2js compiles to JS
  • dart runs in command-line
  • "Dartium" (Chromium+Dart VM) only working browser w/Dart
  • Firefox, IE, Safari will probably never support it
  • wat.

Dart

  • Supports types, without requiring them
  • Object-oriented
  • Operators are methods
  • "isolates" (cf. Erlang processes) for concurrency
  • Code sharing with pub
  • Neat .. cascade operator
  • int is arbitrary size
  • var c = 'String concatenation '"is just adjacency"

TypeScript

“Starts from JavaScript, Ends with JavaScript”

  • Open-source language from Microsoft
  • Typed superset of JavaScript that compiles to plain JavaScript
  • Uses ES6 proposal for class-based object-oriented programming
  • tsc written in TypeScript on CodePlex
  • Optional & inferred type annotations
  • Classes, interfaces, modules
  • Ambient declarations declare types on existing JS code
  • Obligatory arrow syntax x => Math.sin(x)
  • Visual Studio, WebMatrix, emacs, vim, Sublime Text
